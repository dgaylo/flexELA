# flexELA
[![CIntegration Testing](https://github.com/dgaylo/flexELA/actions/workflows/CI_Testing.yml/badge.svg)](https://github.com/dgaylo/flexELA/actions/workflows/CI_Testing.yml)
[![codecov](https://codecov.io/gh/dgaylo/flexELA/graph/badge.svg?token=7ON8EPWTQE)](https://codecov.io/gh/dgaylo/flexELA)

An implementation of [Eulerian Label Advection (ELA)](https://doi.org/10.1016/j.jcp.2022.111560) which allows dynamic storage of the vector source fraction.


- For instructions on installing, see [Installing FlexELA](doc/Building.md).

- For documentation of the API, see [flexela-docs.pages.dev/api](https://flexela-docs.pages.dev/api/).

- For documentation including internal functions, see [flexela-docs.pages.dev/dev](https://flexela-docs.pages.dev/dev/).
